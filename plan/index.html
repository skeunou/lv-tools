Comment ça marche

3 canaux : cet outil = ton plan + tu coches « fait » + log rapide. Strava = la donnée objective (allure, FC, D+, charge, VAM) que Claude va chercher lui-même. Le chat = Claude analyse et écrit le bloc suivant.

Après chaque séance : coche « Fait », mets ton RPE + un ressenti. Pour les séances clés (longue, côte/test, course), ajoute une note. Astuce : mets aussi ton retour dans le titre Strava (RPE + ZP + 1 ressenti), Claude le lit.

Si tu changes la séance (ex : 1h EF au lieu de 40min prescrites), note-le dans le champ « Réalisé » — laisse vide = fait comme prévu. Ça affine l'analyse. Sur les séances structurées (côte, test), le bloc ⌚ Structure Garmin se copie dans Garmin Connect ou intervals.icu.

Dimanche soir : ton agenda sonne (rappel) → tu ouvres Claude → tu dis « check ma semaine ». Claude pulle Strava et te dit, honnête et intransigeant, si la suite du bloc tient.

Fin de bloc (~4 sem) : bouton Export suivi → tu colles dans le chat → Claude croise avec Strava et génère le bloc suivant.

Échelle RPE (effort perçu)

Ressenti global de la séance, noté juste après.

Dans ton plan : EF / longue = 3-4 · côte hebdo = 7-8 · test VAM = 8 · récup = 1-2. Une séance « facile » qui sort à 6+ = signal de fatigue, note-le.

Qui fait quoi

Toi : tu cours · tu coches · titre Strava (RPE/ressenti/ZP) · tu valides · tu signales tout imprévu.
Claude : va chercher les chiffres · analyse · écrit le plan · te dit franchement si un choix bride la progression.
